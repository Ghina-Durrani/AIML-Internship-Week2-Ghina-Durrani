## Dataset Information
This project uses the **Titanic dataset (Kaggle)** which contains information about passengers aboard the Titanic.  
It includes features such as age, sex, passenger class, fare, family size, and survival status.

- Rows: 891  
- Columns: 12  
- Target: Survived (0 = No, 1 = Yes)

---

## Top 3 Survival Insights

1. **Gender was the strongest factor**  
   Females had a much higher survival rate compared to males.

2. **Passenger class mattered a lot**  
   First-class passengers had significantly higher survival chances than third-class passengers.

3. **Fare influenced survival**  
   Passengers who paid higher fares (wealthier passengers) had better survival probability.
